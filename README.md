# Multimedia Processing

This repository contains implementations of various Multimedia Processing techniques developed as part of the Multimedia Processing laboratory coursework.

## Topics Covered

- Quantization
  - Greyscale via Desaturation
  - Median Cut
  - Octree Quantization

- K-Means Rate Distortion Quantization

- Frequency and Spatial Sampling

- High Dynamic Range (HDR) Imaging

- Spatial Filtering
  - Box Filter
  - Gaussian Filter
  - Normalized Gaussian Filter

- Bit Plane Slicing
